What’s New in 5.1

This document contains descriptions of some of the more interesting or significant changes in the WindowTester Pro v5.1 release.

  1. Eclipse 3.6 Support
  2. Improved Widget Locator Support
  3. Better Usability

Eclipse 3.6 Support

WindowTester Pro v5.1 provides full support for Eclipse 3.6 (Helios).

Improved Widget Locator Support

A number of new locators and assertions have been added such as

  • Parameterized Tool Item Locator
  • Pull-Down Menu Item Index State Assertions
  • Pull-Down Menu Item State Assertions

Better Usability

A variety of items have been addressed to improve the user experience including

  • Enhancing support for named combos
  • TreeItem context click reveal and check selections,
  • Combo item recording
  • Playback delay settings
  • International keyboards
  • JLabels in tooltips
