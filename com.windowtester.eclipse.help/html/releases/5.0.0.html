WindowTester Pro v5.0 Highlights

WindowTester Pro version 5.0.0 dramatically improves UI test performance and Linux support, extends the Assertions UI to support Swing and fixes a variety of items to improve usability. Highlights include:

  • UI Test Performance Ð On average, UI test runs 2.5 times faster, and overall performance is significantly enhanced.
  • Linux Support Ð A core SWT runtime re-design provides dramatic improvements in support for Linux.
  • Assertions Inspector Supports Swing Ð Inspector support has been added for Swing recording.
  • Runtime Fixes Ð A variety of items have been addressed to improve the runtime story, including:
    • enhancing key entry and embedded runtime support for Swing,
    • Swedish and US keyboard support,
    • combo item selection fixes, and
    • named tool item enablement assertion support.
