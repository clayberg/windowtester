Simulating Keyboard Events

Keystrokes (such as Ctrl+S) are captured and consumed by a global key event handler to which there is no easy access. As a result these events go unrecognized by the recorder. You can simulate keyboard events by sending key events directly in your test case.

For example:

 	IUIContext ui = ...;
 	ui.keyClick(SWT.CTRL, 'a'); //send ctrl-a  
 	ui.keyClick(SWT.BS);        //send backspace
