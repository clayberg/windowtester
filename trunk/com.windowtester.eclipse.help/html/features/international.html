Non-US Keyboard Support

WindowTester Pro supports testing with Non US Keyboards. We currently have support for the following languages:

  • English
  • German (Germany)
  • Dutch (Belgium)
  • French (France)
  • Spanish (Spain)
  • Swedish (Sweden)
  • Japanese (Japan)
If the language/locale of the keyboard you are using is not currently on the list, with your help, we can add support for it. See the Programmer's Guide for more information.
