New Test Class Wizard

The New Test Class Wizard helps you create a test class for a WindowTester Pro recording.

Option Description Default
Source folder Enter a source folder for the new test class. Either type a valid source folder path or click Browse to select a source folder via a dialog. The source folder of the element that was selected when the wizard has been started.
Package Enter a package to contain the new class. Either type a valid package name or click Browse to select a package via a dialog. The package of the element that was selected when the wizard has been started.
Name Type a name for the new class. <blank>
Modifiers Select an access modifiers for the new class.
  • public
  • default
public
Superclass Select from the dropdown list or click Browse to select a superclass for this class. The SWT test default is com.windowtester.runtime.swt.UITestCaseSWT. The Swing test default is com.windowtester.runtime.swing.UITestCaseSwing. Overridden defaults are persisted.
Which conditions would you like to ensure at setup? Choose conditions to ensure:
  • Workbench has keyboard focus: ensures that the workbench has focus.
  • "Welcome" page is closed: ensures that the "Welcome" page is closed.
  • Workbench is maximized: ensures that the workbench window is maximized.
 Workbench focus and "Welcome" page handlers are enabled. Selection changes are persistent.

Defaults can be restored by clicking on the Default button.
